# For modification
Flutter is Required in Visual Studio Code (or Android Studio!!!)
## instructions to run (windows)
1. Windows+R
2. ms-settings:developers
3. Press Enter
4. Turn Developer Mode on
5. Run Project
<br>
(Developer mode is needed for shared_preferences, which is a type of local storage for Flutter)
<br>
<br>
To Build APK, run in terminal:
<br>
flutter build apk --release or flutter build apk --debug
<br>
flutter build apk --split-per-abi --release
<br>
flutter build appbundle --release (google play)
